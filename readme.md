# Employee Directory 

## by Moses Palacios

##  Description

This is a MERN website that allows users to view employee information at an organization.

## Features
Users can:
* View employee information;
* Sort employees alphabetically and by department name;
* Add new employees

## Bugs

* No known bugs at this time

## Technologies Used
* Mongo DB
* React
* Express
* Node JS
* Mongoose
